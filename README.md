# HousingPricePrediction
## Code Folder

This folder contains the python code used to predict the prices of house with various models:
Gradient Boosting Regressor(GBR), XGBoost(XGB), Random Forest(RF), K-Nearest Neighbour(KNN), AdaBoost(ADA), and Lasso. 
This project was completed in 2 phases. Checkpoint1.py was first phase and final_project.py was the complete project.
## Doc Folder
Contains our documentation and our findings. All the above mentioned models are compared and the most accurate model is selected. 

## Images Folder
Contains the images and screenshots that were used during the documentation.
